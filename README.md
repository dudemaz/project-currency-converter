# Currency Converter

Веб-приложение на чистом JavaScript для конвертации валют с использованием API ЦБ РФ.

## Live Demo

Проект уже задеплоен и доступен по ссылке: [http://64.188.61.83/](http://64.188.61.83/)

## Статус проекта

Проект находится на **финальной стадии**: основной функционал уже реализован и работает стабильно.

Сейчас доступно:
- конвертация валют в реальном времени;
- выбор валют через `Choices.js`;
- swap валют;
- история конвертаций;
- загрузка курсов на выбранную дату;
- кнопка возврата к актуальным курсам;
- загрузочный экран;
- кеширование курсов в `localStorage`.

## Стек

- HTML5
- CSS3
- Vanilla JavaScript (ES Modules)
- [Choices.js](https://github.com/Choices-js/Choices)
- API ЦБ РФ: `https://www.cbr-xml-daily.ru/daily_json.js`

## Структура проекта

- `index.html` — разметка приложения;
- `css-files/` — стили;
- `js/main.js` — точка входа и инициализация;
- `js/api/` — загрузка данных API;
- `js/ui/` — рендер, DOM-элементы, UI-обработчики;
- `js/features/` — фичи (конвертация, история);
- `js/storage/` — работа с `localStorage`.

## Локальный запуск

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/dudemaz/project-currency-converter
   cd "project-currency converter"
   ```
2. Запустить через локальный сервер (например, Live Server в VS Code).
